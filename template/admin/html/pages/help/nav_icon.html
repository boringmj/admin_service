主要讲述如何设置导航的图标。

特别注意：必须要遵守指定的写法

1.默认的图标指向的值layui内置图标

2.要想使用ok-icon图标必须设置navs.json中的fontFamily属性和icon属性

{
   "title": "后台首页",
   "fontFamily":"ok-icon",
   "icon":"ok-iconairplay",
   "href": "pages/home.html",
},
{
   "title": "后台首页",
   "fontFamily":"ok-icon",
   "icon":"&#xe9a6;",
   "href": "pages/home.html",
}
