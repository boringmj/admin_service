主要讲述如何在内部页面添加导航。

特别注意：添加导航的同时必须要遵守指定的写法

1.必须指定一个html元素

2.此元素的属性必须有data-url和lay-id

3.data-url属性表示指定的页面且必须是网站的绝对路径

4.lay-id属性，请不要以数字开头，推荐以字母开头，长度不限

5.当lay-id重复时，那么则只会打开一个，后者只会定位到当前选项卡

   layui.use(function(){
      var okTab = parent.objOkTab;//获取父级的okTab;layui.okTab()不推荐这样获取
      var page = '<div lay-id="add-1" data-url="/pages/user/user.html">个人中心</div>';
      okTab.tabAdd(page);
   })
