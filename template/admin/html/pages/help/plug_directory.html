主要描述插件的目录和插件的引入。

1.如果你想使用okCountUp数字步进器，那么你需要引入一个okCountUp插件

引入插件并加载一个插件
layui.use(['okCountUp'], function () {
	var countUp = layui.okCountUp;
	var okTab = parent.objOkTab;//获取父级的okTab;layui.okTab()不推荐这样获取
	var page = '<div lay-id="add-1" data-url="/pages/user/user.html">个人中心</div>';
	okTab.tabAdd(page);
});
